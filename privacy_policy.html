Privacy Policy

Last updated: March 21, 2025

Information Collection and Use

The Da phuq? Chrome extension doesn't collect any information. Text selected on web pages is sent to OpenAI's API using your own key

Data Storage

Your OpenAI API key is stored on your device in Chrome's local storage. We do not maintain any servers or databases to store your information.

API Usage

This extension uses the OpenAI API to generate explanations. When you request an explanation:

  • The selected text is sent to OpenAI's servers using your provided API key
  • The communication is encrypted using HTTPS

Third-Party Services

We use the following third-party services:

  • OpenAI API - For generating explanations (subject to OpenAI's privacy policy)

Data Security

Your OpenAI API key is stored securely in Chrome's local storage

Text that you select and OpenAI's output is never sent anywhere else.

Changes to This Policy

We may update this privacy policy from time to time. You will be notified of any changes through the extension's update mechanism.
