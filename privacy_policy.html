Privacy Policy

Last updated: March 21, 2025

Limited Use Disclosure

This extension complies with Chrome Web Store's Limited Use requirements as follows:

  • Allowed Use: We only collect and process selected text to provide our core feature: generating simple explanations using the OpenAI API. This is the single purpose of our extension.
  • Allowed Transfer: Selected text is only transferred to OpenAI's API for the sole purpose of generating explanations. No other transfers of user data occur.
  • Prohibited Advertising: We never use the selected text or any user data for advertising purposes. No personalized, re-targeted, or interest-based advertising is ever conducted.
  • Prohibited Human Interaction: No humans have access to the selected text or generated explanations. All processing is done automatically through the OpenAI API.

Information Collection and Use

The Da phaq? Chrome extension collects the following data:

  • Text that you explicitly select on web pages (only when you request an explanation)
  • Your OpenAI API key (which you provide)

This data is used solely for generating explanations through the OpenAI API.

Data Storage

Your OpenAI API key is stored securely in Chrome's local storage on your device. We do not maintain any servers or databases to store your information. Selected text is temporarily processed and immediately discarded after generating the explanation.

API Usage

This extension uses the OpenAI API to generate explanations. When you request an explanation:

  • The selected text is sent to OpenAI's servers using your provided API key
  • The communication is encrypted using HTTPS
  • The text and resulting explanation are used only for providing the explanation service

Third-Party Services

We use the following third-party services:

  • OpenAI API - For generating explanations (subject to OpenAI's privacy policy)

User Consent

Before collecting any text or using the OpenAI API, we require explicit user consent through:

  • Requiring users to manually select text they want explained
  • Requiring users to provide their own API key
  • Requiring users to explicitly request an explanation through the extension interface

Data Security

We implement several security measures:

  • Your OpenAI API key is stored securely in Chrome's local storage
  • All communication with OpenAI's API is encrypted
  • Selected text and explanations are never stored permanently
  • No data is shared with any third parties except OpenAI (for generating explanations)

Data Removal

When you uninstall the extension, Chrome automatically removes all stored data, including your API key. No manual action is required to clear your data.

Changes to This Policy

We may update this privacy policy from time to time. You will be notified of any changes through the extension's update mechanism.
